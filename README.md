# 🛡️ Inteligencia de Amenazas y Visualización Geoespacial
### *Un Enfoque Analítico para la Mitigación del Impacto Económico en Incidentes Digitales*

## 📋 Descripción del Proyecto
Este proyecto fue desarrollado como parte del **Nivel Integrador del programa Talento Tech**. Utiliza ciencia de datos para analizar un dataset global de amenazas cibernéticas (2015-2024), identificando patrones de ataque y su correlación con pérdidas financieras.

## 🚀 Tecnologías Utilizadas
* **Lenguaje:** Python 3.x
* **Base de Datos:** SQLite (SQL)
* **Dashboard:** Streamlit
* **Visualización:** Plotly Express & Graph Objects
* **Control de Versiones:** GitHub

## 📂 Estructura del Repositorio
* `app.py`: Dashboard interactivo de inteligencia.
* `procesar_datos.py`: Script ETL (CSV -> SQLite).
* `requirements.txt`: Dependencias del proyecto.
* `data/`: Contiene la base de datos `ciberseguridad.db`.

## 🛠️ Instalación y Uso
1. Clona el repositorio.
2. Instala librerías: `pip install -r requirements.txt`.
3. Ejecuta la app: `streamlit run app.py`.
