# 🛡️ Análisis Integral de Ciberseguridad Global (2015-2024)

## 📋 Descripción del Proyecto
Este proyecto fue desarrollado como parte del **Nivel Integrador del programa Talento Tech**. El objetivo principal es analizar un dataset global de amenazas cibernéticas para identificar patrones de ataque, impactos financieros y la eficiencia de los mecanismos de defensa modernos.

A través de una arquitectura que combina **SQL** para la gestión de datos y **Python** para la visualización, esta herramienta permite a los tomadores de decisiones visualizar riesgos críticos y adoptar estrategias proactivas.

## 🚀 Tecnologías Utilizadas
* **Lenguaje:** Python 3.x
* **Base de Datos:** SQLite (SQL)
* **Dashboard:** Streamlit
* **Visualización:** Plotly (Gráficos interactivos)
* **Control de Versiones:** GitHub

## 📂 Estructura del Repositorio
* `app.py`: Aplicación principal de Streamlit con el dashboard interactivo.
* `procesar_datos.py`: Script de procesamiento ETL (Extracción, Transformación y Carga) del CSV a la base de datos SQL.
* `consultas_analisis.sql`: Archivo con las consultas SQL avanzadas utilizadas para el análisis.
* `data/`: Carpeta que contiene el dataset original y la base de datos generada.
* `requirements.txt`: Lista de dependencias necesarias para ejecutar el proyecto.
